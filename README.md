# Vasteras Indians Website

Static website for the Vasteras Indians community, designed for direct hosting on GitHub Pages.

## Files

- `index.html` - main landing page
- `styles.css` - responsive styling
- `script.js` - small scroll reveal behavior
- `assets/*.svg` - local demonstrative event illustrations
- `.nojekyll` - ensures GitHub Pages serves the site as plain static files

## GitHub Pages

This project does not require a build step, package manager, or framework.

To publish with GitHub Pages:

1. Push the repository to GitHub.
2. In the repository settings, open `Pages`.
3. Set the source to deploy from the `main` branch and the `/ (root)` folder.
4. If you later connect a custom domain, add it in GitHub Pages settings and create the matching DNS records with your domain provider.
